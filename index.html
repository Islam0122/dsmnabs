Слайд 2 — Основы

Повторяем Pygame

Вспомним основу — что нужно знать перед тем как добавлять очки и жизни.

🪟
Окно игры
Экран — это главный холст. Создаём через display.set_mode((800, 600)). Размер в пикселях.
🕹️
Player (Игрок)
Игрок — это объект pygame.Rect. Прямоугольник с координатами X, Y, шириной и высотой.
⬆️
Движение
Читаем клавиши через pygame.key.get_pressed(). Стрелки или WASD меняют координаты X и Y.
⏱️
FPS (кадры)
60 FPS = плавная игра. Ограничиваем через clock.tick(60). Без этого — лютая скорость!
game.py 
import pygame
pygame.init()

screen = pygame.display.set_mode((800, 600))
clock  = pygame.time.Clock()

running = True
while running:            # 🔄 Главный игровой цикл
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False

    screen.fill((20, 20, 30))    # Заливаем фон
    pygame.display.flip()        # Показываем кадр
    clock.tick(60)               # 60 FPS

Слайд 3 — Концепция

Игровая логика

Правила, победа, поражение — именно это делает игру игрой.

📜
Правила
Что разрешено делать игроку. Куда можно идти. Что нельзя.
🏆
Победа
Набрал 100 очков → WIN! Убил босса → следующий уровень.
💀
Поражение
HP = 0 → Game Over! Экран смерти и возможность начать заново.
⭐
Очки (Score)
Монета → +10. Враг → +25. Уровень → +100. Мотивирует играть!
❤️
Здоровье (HP)
hp = 3. Коснулся врага → hp -= 1. Жизней нет → Game Over.
Примеры из популярных игр
⛏️
Minecraft
10 ❤️ здоровья. Монстр бьёт → -1 HP. Смерть → возрождение и потеря вещей.
🐦
Flappy Bird
Труба = мгновенная смерть. Чем дольше летишь — тем больше очков.
👾
Pac-Man
Точки = +10 очков. Призрак касается = -1 жизнь. 3 жизни всего.
Слайды 4–5 — Score

Система очков Score

Score — это просто число. Когда что-то хорошее происходит — прибавляем. Вот и всё!

1
Создаём переменную
score = 0
Начальное значение всегда 0. Это просто число Python.
2
Добавляем очки
score += 10
+= это shortcut для score = score + 10. Монета → +10!
3
Отображаем на экране
font.render(f'Score: {score}', True, (255,215,0))
Превращаем число в текст и рисуем на экране.
4
Помещаем на экран
screen.blit(text, (10, 10))
blit = "приклеить". (10, 10) — координата левого верхнего угла.
score_system.py 
score = 0
font  = pygame.font.SysFont('Arial', 36)

# В игровом цикле:

# Собрали монету?
if player.colliderect(coin):
    score += 10        # +10 очков!

# Убили врага?
if player.colliderect(enemy):
    score += 25        # +25 очков!

# Рисуем счёт:
text = font.render(
    f'Score: {score}', True, (255, 215, 0)
)
screen.blit(text, (10, 10))

Слайд 6 — Здоровье

Система HP (жизни)

HP = Health Points = жизни игрока. Когда hp = 0 — игра заканчивается. Всё просто!

❤️
Что такое HP?
Это обычное число: hp = 3. Три жизни у игрока в начале игры.
💥
Теряем HP
Столкнулся с врагом → hp -= 1. Минус одна жизнь!
💊
Восстанавливаем HP
Собрал аптечку → hp += 1. Но не больше максимума!
hp = 3 (полный)
❤️❤️❤️
→ Всё хорошо
hp = 1 (опасно!)
❤️🖤🖤
→ Почти умер!
hp = 0 (конец)
🖤🖤🖤
→ GAME OVER!
hp_system.py 
hp     = 3       # Начальные жизни
MAX_HP = 5       # Максимум жизней

# Игрок получает урон:
hp -= 1
print(f"❤️ HP: {hp}")

# Проверяем Game Over:
if hp <= 0:
    print("💀 GAME OVER!")
    running = False

# Игрок нашёл аптечку:
if hp < MAX_HP:
    hp += 1     # Восстанавливаем HP

# Отображаем HP:
hp_text = font.render(
    f'HP: {"❤" * hp}', True, (255, 80, 80)
)
screen.blit(hp_text, (10, 50))
💡 Задание: Добавь условие — если hp > MAX_HP, то hp = MAX_HP. Это не даст игроку иметь больше жизней чем максимум!
Слайд 7 — Столкновения

Логика столкновений

Collision Detection — определяем когда объекты касаются. Pygame делает это через colliderect().

collisions.py 
player_rect = pygame.Rect(x, y, 40, 40)     # Прямоугольник игрока
enemy_rect  = pygame.Rect(ex, ey, 40, 40)   # Прямоугольник врага
coin_rect   = pygame.Rect(cx, cy, 20, 20)   # Прямоугольник монеты

# ──────────────────────────────────────
# Столкновение с ВРАГОМ → теряем HP
# ──────────────────────────────────────
if player_rect.colliderect(enemy_rect):
    hp -= 1                         # Теряем жизнь
    player_rect.x, player_rect.y = 400, 300  # Сброс позиции
    if hp <= 0:
        running = False             # Конец игры!

# ──────────────────────────────────────
# Столкновение с МОНЕТОЙ → получаем Score
# ──────────────────────────────────────
if player_rect.colliderect(coin_rect):
    score += 10                      # +10 очков!
    coin_rect.x = random.randint(0, 760)  # Перемещаем монету
    coin_rect.y = random.randint(0, 560)
💥
Касание врага
colliderect(enemy) → hp -= 1
⭐
Сбор монеты
colliderect(coin) → score += 10
💊
Аптечка
colliderect(health) → hp += 1
🚪
Выход с уровня
colliderect(door) → следующий уровень!

Слайды 8–9 — Проект

Полный код игры

Рабочая игра: игрок собирает монеты (+10 Score), избегает врага (−1 HP). HP=0 → Game Over!

my_game.py — полная игра 
import pygame, random
pygame.init()

# ── НАСТРОЙКА ─────────────────────────────────────
screen = pygame.display.set_mode((800, 600))
pygame.display.set_caption("Моя первая игра!")
clock  = pygame.time.Clock()
font   = pygame.font.SysFont('Arial', 28)

# ── ПЕРЕМЕННЫЕ ИГРЫ ───────────────────────────────
score   = 0
hp      = 3
running = True

# ── ОБЪЕКТЫ (прямоугольники) ──────────────────────
player       = pygame.Rect(400, 300, 40, 40)
coin         = pygame.Rect(random.randint(0,760), random.randint(0,560), 20, 20)
enemy        = pygame.Rect(100, 100, 40, 40)
enemy_speed  = [2, 2]

# ── ГЛАВНЫЙ ЦИКЛ ──────────────────────────────────
while running:

    # 1. СОБЫТИЯ
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False

    # 2. ДВИЖЕНИЕ ИГРОКА (стрелки)
    keys = pygame.key.get_pressed()
    if keys[pygame.K_LEFT]  and player.x > 0:   player.x -= 5
    if keys[pygame.K_RIGHT] and player.x < 760: player.x += 5
    if keys[pygame.K_UP]    and player.y > 0:   player.y -= 5
    if keys[pygame.K_DOWN]  and player.y < 560: player.y += 5

    # 3. ДВИЖЕНИЕ ВРАГА (туда-обратно)
    enemy.x += enemy_speed[0]
    enemy.y += enemy_speed[1]
    if enemy.x < 0   or enemy.x > 760: enemy_speed[0] *= -1
    if enemy.y < 0   or enemy.y > 560: enemy_speed[1] *= -1

    # 4. СТОЛКНОВЕНИЯ
    if player.colliderect(coin):             # Монета!
        score += 10
        coin.x = random.randint(0, 760)
        coin.y = random.randint(0, 560)

    if player.colliderect(enemy):            # Враг!
        hp -= 1
        player.x, player.y = 400, 300
        if hp <= 0: running = False

    # 5. РИСОВАНИЕ
    screen.fill((20, 20, 30))
    pygame.draw.rect(screen, (100, 180, 255), player)    # Игрок синий
    pygame.draw.rect(screen, (255, 215, 0),   coin)      # Монета жёлтая
    pygame.draw.rect(screen, (255, 60,  60),  enemy)     # Враг красный

    # 6. HUD (интерфейс)
    screen.blit(
        font.render(f'Score: {score}', True, (255, 215, 0)), (10, 10)
    )
    screen.blit(
        font.render(f'HP: {"❤" * hp}', True, (255, 80, 80)), (10, 45)
    )
    if hp <= 0:
        screen.blit(
            font.render('GAME OVER!', True, (255, 0, 0)), (300, 250)
        )

    pygame.display.flip()
    clock.tick(60)

pygame.quit()
